# Restaurant Website

My first project — a simple, responsive restaurant landing page.

## Features
- Clean layout with header, menu section, about, and contact area
- Mobile-friendly (responsive) design using media queries
- Only HTML5 + CSS3 (no frameworks or JavaScript yet)

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)

## How to View Locally
1. Clone or download this repo
2. Open `index.html` in any browser
